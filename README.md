## Teminal
- new teminal window and tag
new windows : ctrl + alt + T
new tag : shift + ctrl + T

- switch tag
alt + n

- change default path
asume changing to ~/text/program ,add codes below at the end of ~/.bashrc:
```
if [ -d ~/text/program ]; then
cd ~/text/program
fi
```

then reboot your PC.
